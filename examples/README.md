# Example Outputs

This folder contains example output images generated by the avatar system.

## Files

- `sample_output1.png` - Avatar generated from test_sample.wav (440 Hz sine wave)
- `sample_output2.png` - Avatar generated from test_sample2.wav (523 Hz sine wave)

## How to Generate

```bash
# Generate from test audio
python main.py --audio data/audio_samples/test_sample.wav --output examples/output.png

# Generate from custom audio
python main.py --audio your_audio.wav --output examples/your_output.png
```
